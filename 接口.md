## RAGFlow 项目接口统计结果

经过对项目中所有定义路由的文件进行逐一统计，该项目共有 **约 303 个 API 路由定义**（按 `@manager.route` / `@admin_bp.route` 装饰器计数）。

以下是按模块的详细分布：

### 一、主 API (`/api/v1/`) — restful_apis 目录（共 222 个）

| 文件 | 接口数 | 说明 |
|------|--------|------|
| `agent_api.py` | 30 | Agent/Chat 智能体相关 |
| `dataset_api.py` | 24 | 知识库/数据集管理 |
| `document_api.py` | 19 | 文档上传/解析/管理 |
| `chat_api.py` | 19 | 对话/会话管理 |
| `connector_api.py` | 15 | 数据源连接器 |
| `provider_api.py` | 15 | LLM 模型供应商管理 |
| `user_api.py` | 14 | 用户认证/登录/注册 |
| `memory_api.py` | 12 | 记忆管理 |
| `system_api.py` | 11 | 系统状态/配置/Token |
| `chunk_api.py` | 9 | 文档分块/检索 |
| `bot_api.py` | 9 | 聊天机器人/搜索机器人 |
| `file_api.py` | 7 | 文件管理 |
| `search_api.py` | 7 | 搜索管理 |
| `mcp_api.py` | 7 | MCP 服务器管理 |
| `tenant_api.py` | 5 | 租户管理 |
| `chat_channel_api.py` | 5 | 聊天渠道（飞书/微信等）|

### 三、管理员后台（共 43 个）

| 文件 | 接口数 | 说明 |
|------|--------|------|
| `admin/server/routes.py` | 43 | Admin 后台：用户管理、角色权限、服务管理、沙盒配置等 |

### 汇总

| 分类 | 数量 |
|------|------|
